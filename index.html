The basic difference between flexbox and grid is, flex is one dimensional layout and grid is two dimensional layout. This means flex only can work in column direction or row direction.On the other hand, grid can work in both row and column direction in same time.Flex start with display:flex. It works with all kind of display devices and screen sizes.Grid, on the other hand, start with display:grid. And then we need to set column and row sizes with grid-template-columns and grid-template-rows, and then place its child elements into the grid with grid-column and grid-row.

Media Query is the key part of Responsive design.Media query allows us to set different layout for different device or screen size.It allows us to set unique layout for a specific screen size.As an Example, there are many elements like grid or a particular layout in a landing page that is not suitable for mobile screen viewport.But when we break the grid or layout with media query,it become suitable for mobile devices or a particular screen sizes.

CSS Box model is a container that contains multilple elements like border,margin,padding,and the content itself.CSS box model allows us to create the design and layout of a web page.The content property of CSS box model is used to display all the elements of an web page.Padding is used to create space around the elements inside the border.Border property is used to cover the content & any padding, & also allows to set the style, color, and width of the border.Margin is used to craete space outside & around the border.

Semantic tag is a HTML tag that has a meaning on their own.The semantic HTML tags help the search engines and other user devices to determine the importance and context of web pages. The pages made with semantic elements are much easier to read for both the developer and search engines. It has greater accessibility. It offers a better user experience.Before semantic tag,it was very difficult to get context of an HTML code.But semantic tag made it easy to understand the code when someone has no experience.
